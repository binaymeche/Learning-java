# ☕ Introduction to Java

## 📌 What is Java?

Java is a **high-level, object-oriented programming language** developed by **Sun Microsystems** and officially released in **1995**. It is now owned and maintained by **Oracle Corporation**.

Java is one of the world's most popular programming languages because it is **platform-independent**, meaning a Java program can run on different operating systems without changing the source code.

> **Write Once, Run Anywhere (WORA)** is Java's biggest advantage.

Today, Java powers billions of devices and is widely used to build:

- Desktop Applications
- Web Applications
- Mobile Applications (Android)
- Enterprise Software
- Cloud Applications
- Banking Systems
- Games
- Embedded Systems
- Big Data Applications

---

# 🚀 Features of Java

## 1. Object-Oriented

Java is based on **Object-Oriented Programming (OOP)**, where everything revolves around objects and classes.

Java supports:

- Classes
- Objects
- Inheritance
- Encapsulation
- Polymorphism
- Abstraction

### Example

A **Car** is a class.

Your own **Tesla** is an object of the Car class.

---

## 2. Platform Independent

Unlike C or C++, Java does not compile directly into machine code.

Instead:

```
Java Source Code (.java)
        ↓
     Compiler (javac)
        ↓
     Bytecode (.class)
        ↓
 JVM (Java Virtual Machine)
        ↓
 Operating System
```

Since every operating system has its own JVM, the same bytecode can run everywhere.

This is why Java is called

> **Write Once, Run Anywhere (WORA).**

---

## 3. Easy to Learn

Java has:

- Simple syntax
- English-like keywords
- Automatic memory management

Anyone with basic programming knowledge can learn Java easily.

---

## 4. Secure

Java is designed with security in mind.

Security features include:

- No direct memory access
- Automatic Garbage Collection
- Bytecode Verification
- Runtime Security Checks

These features reduce crashes and memory-related bugs.

---

## 5. Architecture Neutral

Java bytecode is independent of processor architecture.

Whether your computer has:

- Intel
- AMD
- ARM

the JVM executes the same bytecode correctly.

---

## 6. Portable

Java programs can run on:

- Windows
- macOS
- Linux

without changing the source code.

---

## 7. Robust

Java is considered a robust language because it provides:

- Strong type checking
- Compile-time error checking
- Exception Handling
- Garbage Collection

These features help developers create reliable software.

---

## 8. Multithreading

Java allows multiple tasks to run simultaneously.

Example:

A music player can

- play music,
- download songs,
- and update the UI

all at the same time.

Java provides built-in support for threads.

---

## 9. High Performance

Although Java runs through the JVM, it is still fast because of the

**JIT (Just-In-Time) Compiler**

which converts frequently used bytecode into native machine code during execution.

---

## 10. Distributed

Java is widely used to develop distributed applications where multiple computers communicate over a network.

Examples include:

- Banking systems
- Cloud applications
- Online shopping websites

| **Jakarta EE (formerly Java EE)** | Enterprise and Web Applications |
| **Java ME (Micro Edition)** | Embedded and small devices |

# 🎯 Interview / Viva Questions

### 1. Who developed Java?

**Answer:** Sun Microsystems.

---

### 2. Who currently owns Java?

**Answer:** Oracle Corporation.

---

### 3. What does WORA stand for?

**Answer:** Write Once, Run Anywhere.

---

### 4. Why is Java platform-independent?

**Answer:** Because Java source code is compiled into bytecode, which runs on the JVM available for different operating systems.

---

### 5. What is bytecode?

**Answer:** Bytecode is the intermediate code generated by the Java compiler (`javac`) and executed by the JVM.

---

### 6. What is JVM?

**Answer:** JVM (Java Virtual Machine) is the software responsible for executing Java bytecode.

---

### 7. What is JIT Compiler?

**Answer:** JIT (Just-In-Time) Compiler converts frequently used bytecode into machine code during execution to improve performance.

---

### 8. Name four features of Java.

**Answer:**

- Platform Independent
- Object-Oriented
- Secure
- Robust
