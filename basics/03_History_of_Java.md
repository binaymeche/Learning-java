# 📜 History of Java

## 📌 Introduction

Java is one of the world's most popular programming languages. It was developed by **James Gosling** and his team at **Sun Microsystems**.

The project began in **June 1991**, and Java was officially released to the public in **1995**.

The main goal of Java was to create a language that could run on many different electronic devices without modification.

---

# ☕ Timeline of Java

| Year | Event |
|------|-------|
| **1991** | Java project started by James Gosling at Sun Microsystems. |
| **1995** | Java 1.0 officially released. |
| **2006** | Java became open-source under the GNU GPL license. |
| **2010** | Oracle Corporation acquired Sun Microsystems and became the owner of Java. |
| **Today** | Java continues to evolve with new versions released approximately every six months. |

# 🌳 How Java Got Its Name

Java went through several names before becoming "Java."

## 1. Green Team

James Gosling led a group called the **Green Team**.

Their mission was to develop software for electronic devices such as set-top boxes and smart appliances.

Initially, the team planned to improve C++, but due to its complexity and memory usage, they decided to create an entirely new programming language.

---

## 2. GreenTalk

The first name of the language was **GreenTalk**.

- File Extension: `.gt`
- Developed by the Green Team.

Later, the team searched for a better name.

---

## 3. Oak

The language was renamed **Oak**.

James Gosling chose this name because an **Oak tree** stood outside his office.

The oak tree symbolizes:

- Strength
- Stability
- Longevity

Unfortunately, the name **Oak** was already trademarked by another company (**Oak Technologies**), so they had to choose another name.

---

## 4. Java

After brainstorming many names such as:

- DNA
- Silk
- Ruby
- Java

the team selected **Java**.

The name was inspired by **Java Coffee**, which comes from the Indonesian island of **Java**.

James Gosling liked the name because it sounded:

- Unique
- Dynamic
- Easy to pronounce
- Memorable

This became the official name when Java 1.0 was released in **1995**.

---

# 🌍 Write Once, Run Anywhere (WORA)

When Java 1.0 was released, it introduced one revolutionary idea:

> **Write Once, Run Anywhere (WORA)**

This means:

- Write the Java program once.
- Compile it into Bytecode.
- Run it on any operating system that has a JVM.

This feature made Java extremely popular.

---

# 🔓 Java Becomes Open Source

### 2006

Sun Microsystems released most of Java under the **GNU General Public License (GPL).**

### 2007

Almost all Java core libraries became open source.

This helped Java grow rapidly because developers around the world could contribute to it.

---

# 🏢 Oracle Acquires Java

In **2010**, Oracle Corporation purchased Sun Microsystems.

Since then, Oracle has maintained and continuously improved Java.

---

# 📈 Major Java Versions

Instead of memorizing every version, remember these important releases.

| Version | Year | Major Feature |
|---------|------|---------------|
| **Java 1.0** | 1995 | First official release |
| **JDK 1.2** | 1998 | Swing, Collections Framework |
| **Java 5** | 2004 | Generics, Enhanced for-loop, Annotations |
| **Java 8** | 2014 | Lambda Expressions, Streams, New Date-Time API |
| **Java 9** | 2017 | Module System (Project Jigsaw) |
| **Java 11** | 2018 | Long-Term Support (LTS) |
| **Java 17** | 2021 | Long-Term Support (LTS), Sealed Classes |
| **Java 21** | 2023 | Long-Term Support (LTS), Virtual Threads, Record Patterns |

> 💡 **Exam Tip:** Focus on Java 5, Java 8, Java 11, Java 17, and Java 21. These are the most influential versions.

---

# ⭐ Important Milestones

### Java 5 (2004)

Introduced:

- Generics
- Enhanced `for` loop
- Autoboxing & Unboxing
- Annotations
- Varargs

---

### Java 8 (2014)

One of the biggest releases.

Introduced:

- Lambda Expressions
- Streams API
- Functional Interfaces
- Default Methods
- New Date-Time API

---

### Java 9 (2017)

Introduced the **Java Module System**, improving application organization and security.

---

### Java 11 (2018)

- Long-Term Support (LTS)
- Performance improvements
- New HTTP Client API

---

### Java 17 (2021)

- Long-Term Support (LTS)
- Sealed Classes
- Pattern Matching improvements

---

### Java 21 (2023)

Current LTS release introducing:

- Virtual Threads
- Sequenced Collections
- Pattern Matching improvements

---

# 📝 Summary

- Java project started in **1991**.
- Developed by **James Gosling**.
- Originally called **GreenTalk**.
- Later renamed **Oak**.
- Finally became **Java**.
- Released officially in **1995**.
- Java follows **Write Once, Run Anywhere (WORA)**.
- Became open source in **2006**.
- Oracle became the owner in **2010**.
- Major versions include Java **5**, **8**, **11**, **17**, and **21**.
